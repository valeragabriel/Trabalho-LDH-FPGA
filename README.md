# Projeto de Automatização para Agricultura Vertical com FPGA

Bem-vindo ao repositório do projeto de automatização desenvolvido pelos alunos da disciplina de HDL da Universidade Federal de Santa Catarina (UFSC). Este projeto visa criar um sistema inteligente utilizando FPGA para otimizar as condições de crescimento em um ambiente de agricultura vertical.

## Integrantes do Grupo
- Gabriel Gobbo Araujo Valera
- Dariel Arian Acosta

## Visão Geral

O objetivo principal deste projeto é criar um sistema automatizado para um estabelecimento de agricultura vertical, garantindo condições ideais para o crescimento saudável da plantação desejada. As principais funcionalidades incluem:

1. **Controle de Temperatura:** Utilizando sensores, o sistema ajusta a temperatura do ambiente para garantir condições ideais para o desenvolvimento das plantas.

2. **Monitoramento de Umidade:** Sensores de umidade são utilizados para garantir que o solo ou substrato de cultivo mantenha níveis de umidade adequados.

3. **Irrigação Automática:** O sistema controla a irrigação conforme necessário, garantindo que as plantas recebam água na quantidade certa para o seu desenvolvimento.

4. **Controle de Iluminação:** A iluminação é ajustada automaticamente para fornecer o fotoperíodo ideal para o crescimento das plantas.

## Requisitos do Sistema

- Placa de Desenvolvimento FPGA: Cyclone III
- Ferramentas de Desenvolvimento HDL: Quartus II e ModelSim
- Sensores de Temperatura, Umidade 
- Sistema de Irrigação (mangueira de irrigação e bomba)
- Luz Led
